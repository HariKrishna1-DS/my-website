Data Science Pipeline

From Raw Data to
Precise Predictions

01
🧹
Data Cleaning
Dropped irrelevant columns (availability, society, balcony, area_type). Removed 90 null rows. Parsed range values like "1133–1384" into averages.
13,320 → 13,246 rows
02
⚙️
Feature Engineering
Extracted BHK count from size strings. Calculated price_per_sqft. Grouped 1,052 rare localities (≤10 listings) under "OTHER ≤10".
242 unique locations
03
🔭
Outlier Removal
Removed properties with <300 sqft/BHK. Applied mean±std filtering per location. Eliminated BHK anomalies where smaller configs cost more.
13,246 → 7,088 rows
04
🤖
Model Building
One-hot encoded 242 locations. Trained Linear Regression with 80/20 split. Validated using ShuffleSplit cross-validation (10 folds).
R² = 0.847
